HTML

HTML stands for hypertext markup language. It is used to create content in a webpage
It uses a system of tags to tell browsers how to display text, images, links, and other elements on a page. Every website you visit is built, at its core, with HTML.

Core topics in HTML

Document Structure

Every HTML document degins with a Doctype declaration followed by the html tag which contains 2 main tags which are head and body tag.

Tags

Tags come in pairs using opening tags and closing tags for example a paragraph tag p as opening tag and /p as closing tag
